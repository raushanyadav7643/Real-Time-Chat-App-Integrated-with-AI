# 💬 QuickChat – Real-Time Chat Application with AI Integration

QuickChat is a modern, full-stack real-time chat application designed to provide seamless communication with advanced features like AI assistance, file sharing, voice/video calls, and more. It offers a clean and responsive user interface with secure authentication and smooth user experience.

## 🚀 Features

* 🔐 **User Authentication**

  * Secure Sign Up & Login system
  * Using User Name and Password

* 💬 **Real-Time Messaging**

  * Instant messaging with live user status (online/offline)
  * Smooth and responsive chat interface

* 📞 **Voice & Video Calls**

  * One-click voice and video calling functionality
  * Call duration tracking

* 📂 **File Sharing**

  * Send and receive documents, images, and media files
  * Preview shared content inside chat

* 🤖 **AI Assistant**

  * Integrated AI assistant for smart replies and assistance
  * Enhances user experience with intelligent interactions

* 📊 **Interactive Features**

  * Polls, events, and location sharing
  * Emoji support and reactions

* 🎨 **Modern UI/UX**

  * Dark and light mode support
  * Clean, minimal, and user-friendly design

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript (Modern UI)
* **Backend:** Node.js / Express (or your stack)
* **Database:** MongoDB / Firebase (update if needed)
* **Authentication:** Email OTP verification
* **Deployment:** Render / Cloud Hosting
